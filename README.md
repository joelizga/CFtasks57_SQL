# CFtasks57_SQL
## Overview
For Career Foundry assessment used provided data from fictional Rockbuster corporation to design company business strategry. Used SQL to answer questions about revenue gain, rental duration, customer location, and sales figures between geographic regions. Used Tableau to generate visualizations for final presentation.
